## :toolbox: Getting Started

### :gear: Installation

Install via Composer
```bash
composer install
```
Generate Application Key
```bash
php artisan key:generate
```
Migrate Database/Models
```bash
php artisan migrate --seed
```
Run seeder
```bash
php artisan db:seed
```
Install Dependencies
```bash
npm install
```
